About This Bot

This bot receives news via webhooks and automatically broadcasts them to all Telegram groups it has been added to.

How to Use

  1. Add the bot to your Telegram group
  2. Make sure the bot has permission to send messages
  3. Send news to the webhook endpoint

Webhook API Reference

To send news to the bot, make a POST request to /news-webhook with the following JSON structure: 

{
    "secret": "your_webhook_secret",
    "news": {
        "id": "unique_id",
        "title": "News Title",
        "content": "News content...",
        "source": "Source Name",
        "url": "https://example.com/news",
        "image_url": "https://example.com/image.jpg",
        "tags": ["tag1", "tag2"]
    }
}

Bot Commands

  • /start - Initialize the bot
  • /help - Show help information
  • /about - About the bot
  • /status - Check if the bot is working
To configure this bot, set the environment variables: TELEGRAM_BOT_TOKEN, WEBHOOK_URL, and WEBHOOK_SECRET
